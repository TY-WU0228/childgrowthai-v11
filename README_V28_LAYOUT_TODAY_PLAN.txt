ChildGrowth AI V28 - Layout + Today Plan

Based on V27.

Changes:
1. Layout optimization:
   - Large version line is hidden from the main UI.
   - Header becomes compact.
   - On subpages, header shows page title instead of repeating app name/version.
   - Version/debug banner is hidden from customer view.

2. Today Plan:
   - Home now has a one-off daily task section.
   - Parents can add Piano practice, Reading log, spelling, etc.
   - Quick buttons:
     - Piano practice
     - Reading log
   - This is for today-only tasks.
   - Recurring activities should still be added in Profile > Custom Activities.

3. Parent insight:
   - If piano is added, AI parent insight explains how to fit it into the evening.

Usage:
- If you want Janice to practise piano tonight, add it in Home > Today Plan.
- If piano lesson happens every Saturday, add it in Profile > Custom Activities.
