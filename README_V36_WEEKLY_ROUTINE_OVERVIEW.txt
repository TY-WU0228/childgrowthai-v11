ChildGrowth AI V36 - Weekly Routine Overview

Based on V35.

New:
- Adds "一週固定行程總覽" in Profile / 我的.
- Shows Monday to Sunday fixed routines in one place.
- Helps parents avoid duplicate routines.
- Shows possible duplicate warning if same day + same activity + same start time exists.
- Shows free time windows for each day:
  Weekdays use after-school window 3:30 PM - 8:30 PM.
  Weekends use 9:00 AM - 8:30 PM.
- The overview is collapsed after the first 3 days by default; user can expand/collapse.
- Adds duplicate guard when adding a new routine through V35 manager.

Does not delete existing routines or records.

Testing:
1. Add several routines in Profile > 管理固定行程.
2. Return to Profile and check 一週固定行程總覽.
3. Try adding an exact duplicate routine; confirm dialog should warn.
4. Expand the weekly overview and check free windows.
