ChildGrowth AI V57 - Capture + Calendar + Neutral Splash

Based on V55.

New:
1. Universal Capture / 快速記錄
   - Added a big Capture entry on Home.
   - Capture supports:
     Photo, Voice, Text, Schedule.
   - AI/fallback routes captures to:
     School Notice, Teacher Comment, Activity Progress, Homework, Routine.
   - Parent must Confirm Add before saving.
   - Existing detailed functions remain; Capture is a shortcut, not a replacement.

2. Calendar Reminder export
   - Routine rows can export .ics calendar events with alarms.
   - School Notice rows can export .ics calendar events with alarms.
   - Today Tasks can export .ics calendar events.
   - Designed as beta-friendly iPhone Calendar reminder pathway.
   - This is not yet push notification; it uses Calendar import.

3. Neutral cover screen
   - Replaced flower-heavy purple splash with a more neutral geometric / growth style.
   - Logo uses "CG" instead of flower emoji.
   - Slogan: 陪孩子成長，看見每一步 / Capture less. Remember more.
