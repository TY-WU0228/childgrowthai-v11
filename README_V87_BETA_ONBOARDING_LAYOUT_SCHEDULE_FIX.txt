ChildGrowth AI V87 - Beta Onboarding Layout + Schedule Fix

Based on V86.

Fixes:
1. Beta onboarding checkbox/radio cards:
   - Text no longer overflows outside the card on iPhone.
   - Each option is now a safe 2-column mobile layout: checkbox + wrapping text.
   - Applies to Step 2 goals, Step 4 report preference and Step 5 privacy notice.

2. Schedule / activity onboarding:
   - Replaced the large free-text activity textarea with structured activity cards.
   - Parent can add one activity at a time:
     Day / Activity / Start / End / Prepare.
   - Quick chips added for Swimming, Soccer, Chinese, Piano and Chess.
   - This is easier for parents and better for future reminders.

3. Data safety:
   - No automatic data deletion.
   - Existing data is preserved.
   - Beta "Start as new family" remains user-controlled only.
