ChildGrowth AI V61 - Smart Prep Suggestions

Based on V60.

New:
- When adding or editing a fixed routine, the app can auto-suggest preparation items based on the activity name.
- Example:
  Basketball -> 水樽、運動鞋 / basketball shoes、毛巾、snack、替換衣服
  Swimming -> 泳衣、毛巾、goggles、水樽、拖鞋、膠袋放濕衫
  Piano -> 琴書、功課簿、鉛筆、記錄本、水樽
- It also suggests homework impact:
  sports usually -> 輕量功課日
  piano/chess/chinese/coding -> 正常

Usage:
- Type an activity name such as Basketball.
- The prep box will auto-fill if it is empty.
- You can edit the prep list manually.
- Press “AI 自動建議準備物品” to re-apply suggestions.
