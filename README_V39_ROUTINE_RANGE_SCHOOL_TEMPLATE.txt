ChildGrowth AI V39 - Routine Range + School Template

Based on V38.

Fix / New:
1. Multi-day routine support:
   - Users can now add Monday-Friday school once.
   - Supported text examples:
     星期一至星期五 8:45 到 3:30 All Prime School
     Monday to Friday 8:45am to 3:30pm school
     weekdays 8:45 to 3:30 school

2. New quick button:
   - Mon–Fri School
   - Adds 5 routines:
     Monday-Friday, 8:45 AM - 3:30 PM, All Prime School.

3. Text parser improvement:
   - Understands weekday ranges and school/返學/all prime.
   - "讀取文字" shows the detected days/time/activity.
   - Press "加入固定行程" to create all days at once.

4. Duplicate protection:
   - Existing same day + same activity + same start time is skipped.

5. Overview note:
   - Explains that weekly overview only shows routines actually added.
   - If Wednesday swimming is expected, it must be added as a fixed routine first.

No data deletion.
Existing routines and records are preserved.
